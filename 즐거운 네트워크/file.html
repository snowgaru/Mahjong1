멘젠 쯔모 <멘젠 한정> : 멘젠, 즉 울지 않은 상태(치,퐁,깡을 안한 상태) 에서 쯔모해서(뽑아서) 화료하면 성립

리치 <멘젠 한정> : 멘젠, 즉 울지 않은 상태(치,퐁,깡을 안한 상태) 에서 텐파이(패가 하나만 더 나오면 나는 상태) 상태가 완성되면 1000점 지불하고 선언 가능. 선언 이후 손패를 바꾸지 못하고, 뽑은 패는 쯔모로 화료가 되지 않으면 버려야만 한다. 왕패(쓰이지 않는 패산)에서 아래에 깔려있는 패를 뒤집어서 뒷도라로 쓴다. 도라에 대해서는 아래의 도라 문단을 참조. 누군가가 리치를 선언하고 지불했던 1000점은 화료한 사람이 가져간다. 즉 2명 이상이 리치를 선언한 상태였다면 화료한 사람이 나와있는 리치봉을 다 가져간다. (리치 선언으로 걸어둔 점수를 다 가져간다는 의미이다) 패가 바닥날 때까지 아무도 화료하지 못해 유국되었다면, 지불했던 점수는 돌려받지 못하고 이월(?)되며, 누군가 화료하여 모두 가져갈 때까지 계속 적립된다.

핑후(평화) <멘젠 한정> : 모든 몸통이 슌쯔(이어지는 숫자)로 이루어져 있고 양면대기로 화료시 성립. 머리는 숫자패 또는 객풍패(자풍, 장풍이 아닌 동남서북)일때만 성립. 마지막 몸통을 만들때, 양면으로 대기해야만 성립한다. 예를 들면 마지막 몸통 후보로 2-3삭을 모아놓고, 대기패로 1삭 4삭을 기다릴 때 같은 경우만 성립한다. 1삭과 3삭을 가지고 2삭을 기다리거나, 1-2삭을 가지고 3삭을 기다릴 경우는 양면으로 대기하는 것이 아니므로 성립하지 않는다.

이페코(일배구) <멘젠 한정> : 똑같은 모양의 슌쯔(3개 연속된 숫자)가 2개있을 경우 성립. 아래 예시에서는 똑같은 3-4-5삭의 슌쯔가 2세트 있으므로 이페코이다.

탕야오(단요구) : 요구패(1,9, 풍패(동남서북), 삼원패(백발중))를 사용하지 않고 화료시 성립. 즉 숫자패의 2~8의 패만 쓸 수 있다.

영상개화(린샹카이호) : 똑같은 패가 4개 있거나, 3개가 있는데 다른 사람이 버려줄 경우 깡을 선언하고 4개를 묶어서 커쯔(트리플)로 쓸 수 있다. 이 경우 부족한 1개의 패를 추가로 영상패(이번 판에서 안 쓰이는 패)에서 가져오는데 이걸로 화료할 경우 성립. 유명하지만 우연히 나는 역이고 확률도 낮으며, 점수도 1판(1점)이므로 실제로는 그냥 드물게 보는 보너스 개념이다.

창깡(창공) : 다른 플레이어가 소명깡(퐁으로 커쯔, 즉 트리플을 울어놨는데 쯔모해서(뽑아서) 1개가 더 나오면 깡을 선언하고 4개로 합칠 수 있다)을 선언했을때 그 패가 자신의 화료패인 경우 론을 부를 수 있다.

해저모월(하이떼이마오위에) : 마지막에 남은 패를 뽑아서 쯔모 화료할 경우 성립. 바로 아래의 "하저로어"와 이름이 비슷하니 혼동하지 않게 유의. 영상개화보다는 확률이 높긴 하지만 마찬가지로 운에 의존하는 역이므로 노리고 하는 것은 좋지 않다. 게다가 마지막 패를 자신이 뽑는다는 보장도 없다.

하저로어(하이떼이후리코미) : 마지막으로 버려진 패를 론으로 화료했을 경우 성립. 바로 위의 "해저모월"과 이름이 비슷하니 혼동하지 않게 유의.

일발 <리치 한정> : 리치한 후 1바퀴 안에 아무도 울지 않고 + 화료가 가능할 시 성립. 멘젠쯔모 + 리치 + 일발만으로 아무 역 없이도 3판까지 가능한데 이를 리즈모라고 줄여 부른다.

<삼원패> : 백 발 중 3가지 중 아무 것이나 하나를 커쯔(트리플)로 모으면 성립. 아래의 예시에는 발을 2장 모은상태에서 퐁으로 가져와서 발을 3장 모아서 난 상태이다.

<자풍패> : 자신이 위치한 방향과 같은 방향패를 커쯔(트리플)로 모으면 성립. e.g) 남(南)에 위치했을 때 南을 3개 모으면 성립. 작혼을 기준으로는 본인 앞에 써 있는 방위(동남서북)에 해당하는 패를 모으면 된다.

<장풍패> : 장(동국, 남국)과 같은 방향패를 커쯔(트리플)로 모으면 성립. 동풍전일 경우에는 東을 3개 모으면 성립. 동풍전(가장 짧은 게임)기준으로 東은 항상 장풍패이다. 반장전(동장전 후 남장전을 이어서 함)에서는 후반에 열리는 남국에서는 南이 장풍패가 된다. 어지간하면 서장전 북장전을 볼일은 없으므로 이들이 장풍패인 경우는 매우 드물다. 작혼에서는 가운데에 동1국, 남3국 같은 식으로 표시되니 그걸 모으면 된다.

더블 리치(다브리) <멘젠 한정> : 게임시작후 1턴만에 리치를 선언하고 화료하면 성립. 단, 자신의 리치 선언 이전에 누군가가 후로(치,퐁,깡)를 했다면, 자신에겐 첫번째 턴이라고 하더라도 더블리치로 인정하지 않는다.

삼색동순 <후로 시 1판 감소> : 만패, 삭패, 통패의 3가지 종류의 패로 같은 슌쯔(연속)를 만들면 성립. 아래의 예에서는 2-3-4의 슌쯔가 만패, 삭패, 통패에 모두 있으므로 삼색동순이다.

삼색동각 : 만패, 삭패, 통패의 3가지 종류의 패로 같은 커쯔(트리플)를 만들면 성립. 깡쯔(4개를 깡을 선언하고(울어서) 공개)의 경우도 커쯔로 취급한다. 아래의 예에서는 5의 커쯔가 만패, 삭패, 통패에 모두 있으므로 삼색동각이다.

찬타(혼전대요구) <후로 시 1판 감소> : 머리 1개와, 몸통 4개에 요구패(1,9, 풍패(동남서북), 삼원패(백발중))가 1개라도 모두 들어가있으면 성립. 아래의 예에서는, 몸통에 백으로 커쯔(3개), 동으로 커쯔(3개), 1-2-3 만패 슌쯔(1만이 1개 있음), 7-8-9 삭패 슌쯔(9통이 1개 있음)로 조건을 만족. 머리에는 1의 통패가 2개 있으므로 마찬가지로 조건을 만족한다. 따라서 찬타가 성립된다.

일기통관 <후로 시 1판 감소> : 한 가지 종류의 패로 123/456/789 모두 모으면 성립. 같은 종류로 123456789를 모두 가지고 있다고 해도 1-2-3 / 4-5-6 / 7-8-9 순으로 몸통을 만들지 못하면 성립하지 않는다. 즉, 치로 울었을 경우 2-3-4나 6-7-8등으로 울면 성립하지 않는다. 3개의 슌쯔 외에는 제약이 없으며, 화료하려면 아무 몸통 1개와 머리 1개가 더 필요하다. 아래의 예에서는 1~9의 통패로 1-2-3, 4-5-6, 7-8-9 슌쯔 3개가 성립되었고, 몸통과 머리가 1개씩 더 있으므로 화료가 된다.

또이또이(대대화) : 커쯔(트리플)만으로 몸통을 구성하면 성립. 즉 트리플만 4개 + 머리 1개를 모으는 역이다.

산안커(삼암각) : 안커(퐁하지 않고(울지 않고) 모아서 만든 트리플)를 3세트 만들면 성립. 론으로 날 경우, 버림패로 만들어진 커쯔는(즉 마지막에 가져와서 3번째의 트리플이 되었다면) 안커가 아니므로 (론도 상대의 패를 가져오는 것이니) 성립되지 않는다. 주의해야 할 점.

치또이(칠대자) <멘젠 한정> : 머리(더블)를 7쌍 모으면 성립. 그래서 '히드라'라고 농담하기도 한다 국사무쌍과 함께 3 3 3 3 2 형태가 아닌 경우에도 화료가 성립하는 특수한 경우이다

혼노두(혼로토) : 수패의 1, 9, 자패(동남서북 백발중)만으로 화료시 성립. 찬타는 몸통 4개 / 머리 1개에 최소 1개씩이고, 혼노두는 14개 모든 패여야 한다는 점이 다르다.

소삼원 : 백, 발, 중 3가지 중 2가지를 커쯔(트리플)로 모으고 하나를 머리(더블)로 만들면 성립. 아래의 예에서는 백이 3장, 중이 3장, 발이 2장으로 몸통 2개, 머리 1개가 성립되었다.

산깡즈(삼공자) : 깡을 3번 치고 화료시 성립.

량페코(이배구) <멘젠 한정> : 이페코를 2쌍을 만들면 성립. 아래의 예에서는 2-3-4만 슌쯔가 2개 있고(첫 번째 이페코), 3-4-5통 슌쯔가 2개 있으므로(두 번째 이페코) 량페코가 성립한다. 머리에는 제약이 없다.

준찬타(순전대요구, 준쨩) <후로 시 1판 감소> : 머리, 몸통에 노두패(1,9)가 모두 1개씩이라도 들어가있으면 성립. 찬타는 자패(동남서북 백발중)가 인정되지만, 준찬타는 그렇지 않다.

혼일색 <후로 시 1판 감소> : 한 종류의 수패(1~9), 그리고 자패(동남서북 백발중)만으로 이뤄지면 성립. 수패의 종류는 만,삭,통중에 아무 것이나 써도 상관이 없다.

청일색 <후로 시 1판 감소> : 한 종류의 수패(1~9)만으로 몸통 4개 머리 1개를 만들면 성립. 혼일색과 마찬가지로 수패의 종류는 만,삭,통중에 아무 것이나 써도 상관이 없다. 여기에 자패가 포함되면 바로 위의 혼일색이 된다.


천화 : 게임의 첫 플레이어가 처음 분배된 13개의 패 + 쯔모로 뽑은 패로 바로 화료가 가능 할 경우 성립한다. 따라서 오야(부모), 즉 동에 위치한 플레이어만 가능하다.

지화 : 게임의 첫 번째 바퀴에 처음 분배된 13개의 패 + 쯔모로 뽑은 패로 바로 화료가 가능 할 경우 성립. 앞에서 아무도 울지 않았어야 한다. 즉 게임의 2번째~4번째 버림패까지(여기까지 하면 1바퀴가 되니까) 쯔모로 나면 지화로 인정된다. 따라서 남, 서, 북에 위치한 플레이어만 가능하다. 이것만 빼면 천화와 동일.

쓰안커(사암각) <멘젠 한정> : 안커(퐁하지 않고 만든, 즉 쯔모로 뽑아서 만드는 트리플)을 손에 4쌍 가지고 있으면 성립. 국민역만 1 론으로 날 경우, 마지막에 줍는 론패로 만들어진 커쯔(트리플)는 쯔모로 모은것이 아니므로(론도 상대의 패를 가져오는 것이니) 성립되지 않는다.[28] 주의! 론으로 나고 싶다면 4개의 커쯔를 쯔모를 통해 (본인이 뽑아서) 몸통으로 만들어놓고, 머리를 론으로 가져와서 화료해야 한다. 론이기 때문에 당연히 론 당한 사람 혼자서 점수를 지불한다.

국사무쌍 <멘젠 한정> : 모든 요구패(만,삭,통패의 1,9, 동남서북 백발중)을 하나씩 총 13종 갖추고, 13가지중 아무거나 2개를 가지고 있다면 성립한다. 국민역만 2 치또이 (머리만 7개 모으기)와 함께 2-3-3-3-3의 규칙을 어기는 두 가지의 특수패이다. 아래의 예에서는 만, 삭, 통패의 1, 9와 동남서북 백발중이 모두 1개씩 있어서 13종류를 다 모았고, 발이 2장이므로 국사무쌍이 된다. 아래의 예에서는 발이 2개지만, 다른 12종류의 패도 아무 것이나 2개였다면 어떤 것이든 상관 없다.

대삼원 : 백 발 중 3종류를 모두 다 커쯔(트리플)로 모으면 성립. ~국민역만 3 부족한 머리 한개와 몸통 하나는 머리/몸통 조건만 맞으면 된다.

자일색 : 자패(동남서북 백발중)만으로 화료하면 성립. 단어 그대로 자패로 일색을 하면 된다. 3 3 3 3 2 형태 외에도 치또이 (2 2 2 2 2 2 2) 형태로도 인정된다.

녹일색 : 2삭, 3삭, 4삭, 6삭, 8삭, 발패 만으로 몸통 4개와 머리 1개를 모아 화료시 성립. 삭패를 모았다고 해도 위의 패 이외의 삭패가 섞이면 그냥 혼일색이 된다. 주의! 조금이라도 패에 녹색이 아닌 색이 섞여있으면 녹일색이 못 된다고 생각하면 된다. 참고로 나무위키의 삭수패 이미지는 파란색이지만 실제 삭수패는 녹색이다. 이름이 잘못된 것이 아니다.

청노두 : 노두패(1,9) 만으로 몸통 4개와 머리 1개를 모아 화료시 성립.

소사희 : 동 남 서 북 4가지 중 3가지를 커쯔(트리플)로, 하나를 머리(더블)로 만들면 성립.
